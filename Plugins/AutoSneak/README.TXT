AutoSneak for hMod (Created by Apropos <Apropos.of.nothing@gmx.com>)
========================

This plugin requires build 126+ of hMod.

[1.0] Installation Instructions

========================
[1.0]Installation Instructions
========================
Place AutoSneak.jar into Plugins folder. Add "AutoSneak" to plugins line of server.properties.
Add /sneak to the commands of any user/group you wish to be able to toggle sneaking.
Add /autosneak to the commands of any group you want to automatically be set to sneaking
when they login. Note that any groups inheriting a group that has the '/autosneak' command
will also be autosneaking. Also note that unless an autosneaking group has access to the
/sneak command, they will not be able to disable sneaking.
